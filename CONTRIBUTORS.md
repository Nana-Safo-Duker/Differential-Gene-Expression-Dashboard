## How to Be Listed

Contributors are added to this list when they:
- Submit a merged pull request
- Report a significant bug that leads to improvements
- Contribute substantial documentation
- Provide significant testing feedback

To add yourself or nominate someone, please:
1. Open a pull request with the addition
2. Include a brief description of the contribution
3. Link to relevant issues or PRs
